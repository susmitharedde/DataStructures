## Difference Between List and Tuple

| List | Tuple |
|------|------|
| Mutable | Immutable |
| Uses [] | Uses () |
| Can modify elements | Cannot modify elements |

## When to Use Dictionary?
Dictionary is used when data is stored in key-value pairs.

Example:
```python
student = {
    "name": "Susmitha",
    "branch": "ECE"
